# Wireframes

Mid-fidelity wireframes for the four key screens of the end-to-end Agentic Developer Portal workflow. SVG-based for vector clarity, easy review in any browser, and a small footprint in version control.

| # | Screen | File |

| 1 | Repository List | [`01-repository-list.svg`](./01-repository-list.svg) |
| 2 | Repository Overview | [`02-repository-overview.svg`](./02-repository-overview.svg) |
| 3 | Agent Task Selection | [`03-task-selection.svg`](./03-task-selection.svg) |
| 4 | Agent Execution Panel | [`04-agent-execution.svg`](./04-agent-execution.svg) |

Open any `.svg` file directly in a browser, IDE preview, or design tool. All wireframes share the same dark color palette and design tokens defined in [`src/index.css`](../../src/index.css).
